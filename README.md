## How to deploy on GitHub Pages

1. Go to your repository → **Settings → Pages**
2. Under **Source**, select **Deploy from branch**
3. Choose **main** branch → **/ (root)** → Save
4. Your portfolio will be live at `https://asmitapatwe.github.io`
